Расшифровка переменных:

- `total_users` – [Число] – Общее количество пользователей
- `page` – [Число] – Номер текущей страницы

- `per_page` – [Число] – Количество пользователей на страницу
- `users` – [Массив] – Информация о пользователе

    - `dir_posts` – [Строка] – Количество добавленных материалов в модуле "Каталог сайтов"
    - `msn` – [Строка] – Значение поля "MSN"
    - `state` – [Строка] – Значение поля "Штаты"
    - `email` – [Строка] – Значение поля "Email пользователя"
    - `user` – [Строка] – Никнейм пользователя
    - `yahoo` – [Строка] – Значение поля "Yahoo"
    - `uid` – [Строка] – ID пользователя
    - `name` – [Строка] – Полное имя пользователя
    - `blog_posts` – [Строка] – Количество добавленных материалов в модуле "Блог"
    - `video_posts` – [Строка] – Количество добавленных материалов в модуле "Видео"
    - `signature` – [Строка] – Подпись пользователя
    - `board_posts` – [Строка] – Количество добавленных материалов в модуле "Доска объявлений"
    - `reg_date` – [Строка] – Дата регистрации пользователя на сайте
    - `title` – [Строка] – Титул пользователя
    - `banned` – [Строка] – Забанен ли пользователь? (yes/no)
    - `ip` – [Строка] – IP пользователя при регистрации
    - `news_posts` – [Строка] – Количество добавленных материалов в модуле "Новости сайта"
    - `avatar` – [Строка] – Ссылка на аватар пользователя
    - `icq` – [Строка] – Значение поля "ICQ"
    - `phone` – [Число] – Номер телефона пользователя
    - `group` – [Массив] – Информация о группе пользователя
        - `name` – [Строка] – Название группы, куда входит пользователь
        - `id` – [Строка] – ID группы
    - `birthday` – [Строка] – Дата рождения пользователя
    - `gender` – [Строка] – Пол пользователя
    - `country` – [Строка] – Страна пользователя
    - `load_posts` – [Строка] – Количество добавленных материалов в модуле "Каталог файлов"
    - `photo_posts` – [Строка] – Количество добавленных материалов в модуле "Фотоальбомы"
    - `faq_posts` – [Строка] – Количество добавленных материалов в модуле "FAQ (вопросы/ответы)"
    - `com_posts` – [Строка] – Количество добавленных комментариев
    - `banrate` – [Строка] – Текущий рейтинг пользователя (в процентах)
    - `forum_posts` – [Строка] – Количество добавленных сообщений в модуле "Форум"
    - `home_page` – [Строка] – Значение поля "Домашняя страница (вебсайт)"
    - `rank` – [Строка] – Текущий ранк пользователя
    - `email_verified` – [Строка] – Подтвержден ли email пользователя? (yes/no)
    - `status` – [Строка] – Текущий статус пользователя на сайте (online/offline)

#### [PUT] Редактор пользователя

> Обязательные параметры: `user_id`

Пример запроса:

    $response = $request->put('/users', array(
        'user_id' => '1',
        'home_page' => 'api.ucoz.net',
        'signature' => 'Подпись из uAPI',
        'icq' => '123456',
        'options' => '1,2,3,4,5,6'
    ));

Возможные параметры для передачи:

- `home_page` – [Строка] – Ссылка на WWW сайт пользователя
- `signature` – [Строка] – Подпись пользователя
- `aol` – [Строка] – AOL пользователя
- `yahoo` – [Строка] – Yahoo пользователя
- `icq` – [Число] – ICQ пользователя
- `avatar` – [Строка] – Аватар пользователя (необходима загрузка изображений)
- `full_name` – [Строка] – Полное имя пользователя (только для локальных пользователей)
- `email` – [Строка] – email пользователя (только для локальных пользователей)
- `birthday` – [Строка] – день рождения пользователя (дата) (только для локальных пользователей)
- `gender` – [Число] – пол пользователя (1 – мужской/2 – женский) (только для локальных пользователей)
- `password` – [Строка] – пароль пользователя (только для локальных пользователей)
- `options` – [Число] – опции пользователя (1 – скрывать email, 2 – Принимать письма от других пользователей сайта, 3 – Отключить показ аватаров в сообщениях форума, 4 – Отключить показ подписей в сообщениях форума, 5 – Получать почтовые извещения о личных сообщениях, 6 – Отключить прием уведомлений об изменении репутации) (разделитель запятая)
- `phone` – [Число] – Номер телефона пользователя
- `yahoo` – [Строка] – Yahoo идентификатор
- `jabber` – [Строка] – Jabber идентификатор
- `skype` – [Строка] – Skype идентификатор

#### [POST] Смена логина пользователя

> Обязательные параметры: `user_id`, `login`

Пример запроса:

    $response = $request->post('/users/chlogin', array(
        'user_id' => '277', //айди пользователя
        'login' => 'NewAPIname' //новый логи пользователя
    ));

Расшифровка переменных для запроса:

- `user_id` – [Число] – АЙДИ пользователя
- `login` – [Строка] – Новый никнейм пользователя (min – 2 символа, max – 25 символов)

Пример ответа:

    {
       "success":{
          "msg":"Новый логин: NewAPIname"
       }
    }

#### [POST] Смена группы пользователя

> Обязательные параметры: `user_id`, `gid`

Пример запроса:

    $response = $request->post('/users/chgroup', array(
        'user_id' => '277', //айди пользователя
        'gid' => '2' //переводим в группу с айди 2 (проверенные)
    ));

Расшифровка переменных для запроса:

- `user_id` – [Число] – АЙДИ пользователя
- `gid` – [Число] – АЙДИ группы пользователя, куда переводить пользователя

Пример ответа:

    {
       "success":{
          "msg":"Изменения сохранены"
       }
    }

#### [POST] Смена пароля локального пользователя

> Обязательные параметры: `user_id`, `password`

Пример запроса:

    $response = $request->post('/users/chpass', array(
        'user_id' => '277', //айди пользователя
        'password' => 'newpassword' //устанавливаем новый пароль пользователю
    ));

Расшифровка переменных для запроса:

- `user_id` – [Число] – АЙДИ пользователя
- `password` – [Число] – новый пароль (min 2 символа, max 15 символов)

Пример ответа:

    {
       "success":{
          "msg":"Пароль успешно изменен"
       }
    }
